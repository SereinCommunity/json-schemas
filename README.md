## 使用方法

1. 新建一个JSON文件
2. 复制如下文本到该文件中
3. 将`$schema`的值替换为上面的链接

```jsonc
{
    "$schema": "https://sereincommunity.github.io/json-schemas/v2/plugin-info.json",
}
```
